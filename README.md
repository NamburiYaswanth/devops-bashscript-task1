

 Calculator Script**

This script takes two numbers and an operator (`+`, `-`, or `*`) from the user.
It performs the calculation using a `case` statement and displays the result.
Helps understand arithmetic and conditional logic in Bash.

 File Count Script

Counts the total number of files and folders in the current directory.
Uses `ls` and `wc -l` commands to get the count.
Simple script to practice command-line utilities.

Greeting Script

Continuously asks for names and greets each person until the user types “done.”
Demonstrates the use of loops and conditional statements.
Useful for understanding interactive input handling.

Temperature Converter Script

Converts temperature from Celsius to Fahrenheit using the formula `F = C × 9/5 + 32`.
Takes user input and prints the converted result.
Shows how to use arithmetic operations with decimal precision in Bash.
